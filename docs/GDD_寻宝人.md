# 游戏设计文档（GDD）
# 《寻宝人》— 文物修复 · 寻宝 · 传承

> **版本**：0.1（概念验证阶段）
> **引擎**：Cocos Creator 3.8.8
> **目标平台**：Android / iOS / 微信小游戏（竖屏，720×1280）
> **核心受众**：18~35 岁，对中国历史文化有基本兴趣，喜欢解压类/放置类游戏的城市用户
> **情感定位**：让玩家在"乐"中"不由自主地思考"——绝不说教，只是好好讲一个故事

---

## 一、核心理念（Design Philosophy）

> "每一件文物，都曾在某个历史的瞬间被人放下，被泥沙掩埋，被人遗忘。
> 你的手指划过屏幕的每一下，都是在和历史重新握手。"

这不是一个"弘扬爱国主义"的游戏。
这是一个关于**好奇心、手艺、选择与传承**的故事。
民族情感是自然生长的结果，不是预设的前提。

---

## 二、故事起点（Opening — 放低基调）

### 世界观
你是一个刚失业的年轻人，在大城市混了几年，浑浑噩噩地回到了外公留给你的沿海小城。

外公的那间**"陈记古玩铺"**，你小时候觉得又闷又臭，满是灰尘和奇怪的气味。但现在，它是你唯一的遗产。

房东来催租了。你翻遍了铺子，只找到一个裹满泥巴的"破罐子"——外公的标签上写着：**"海捞，未清，估值未知"**。

你不知道里面是什么。你没有别的办法。你就这样，开始刮了第一下。

> **设计意图**：起点是"穷"，不是"使命感"。
> 玩家的第一驱动力是**"我需要钱付房租"**，而不是"我要保护中华文明"。
> 家国情怀是结局，不是开场白。

### 核心支撑角色（小人物，真性情）
- **外公（陈老爷子）**：以日记/录音形式出现。语气絮叨，时常说错话，会把自己的粗鄙的捡漏经历吹得天花乱坠——但每隔一段时间，会说出一句让你沉默片刻的话。
- **房东王姐**：刀子嘴豆腐心，时常催租，后来会偷偷帮你在街坊邻里宣传，带来第一批"送宝"的客人。
- **周教授**：退休的文物鉴定专家，邋遢、古怪、对你颐指气使，但是真正的权威。他是你理解文物背后故事的"读书人"。
- **阿涛**：一个开摩托的快递小哥，机缘巧合成为你的"捞宝搭档"，负责开船（虽然经常晕船）。

## 三、核心玩法循环（Core Game Loop）

### 完整循环图（文字版）

```
[出海打捞/收购盲盒] → [刮擦清洁/文物修复] → [专家鉴定/真伪判断]
         ↑                                              ↓
  [资金/声望回收]  ←  [博物馆捐献/拍卖出售/私下交易]  ←  [文物交锋/抉择]
```

## 四、五幕玩法详细设计

### 幕一：获取（Discovery）

#### 玩法：盲盒采购与海捞探索
玩家通过两条路径获取"未清理文物"：

**路径 A — 线下收购（早期主要来源）**
- 每天刷新的"民间送宝"系统：NPC 登门，带来他们的"宝贝"。
- 价格由玩家议价（一个简单的砍价小游戏）。
- NPC 的物品参差不齐：可能是真品、可能是赝品、可能是有趣的生活物件（喜剧性）。

> **鉴宝节目彩蛋**："微波炉专用"梗 — 某大妈拿来一个金光闪闪的"汉代铜镜"，清洗后发现底部有"Made in 1992, Foshan"的字样。这件道具可以放进店里当装饰，每天带来少量金币（大妈来拍照打卡）。

**路径 B — 出海打捞（中期核心来源）**

- 地图以**"外公的手绘航海日记"**形式呈现，充满手写标注和潦草的地图。
- 不同海域/区域对应不同历史时期的沉船和文物（南海 = 宋元瓷器，福建沿海 = 郑和相关，等等）。
- 出海消耗"燃料"（限制频率），并触发**随机叙事卡片事件**（文字+插画风格）：
  - "阿涛说他看到了海底的灯光，你选择：A. 深潜查看 B. 立刻上浮"
  - "一艘挂着外国旗帜的考察船出现了……"

---

### 幕二：修复（Restoration）— 核心 Scratch 玩法

#### 三阶段修复流程

**阶段 1：粗清洁（Core Scratch）**
- 玩家用手指在屏幕上滑动，用工具刮除表面包裹的泥土、铁锈、藤壶等。
- 不同材质对应不同工具和手感：
  - 泥土 → 软毛刷（轻柔，细沙沙声）
  - 铁锈 → 砂纸（粗糙，需要反复摩擦才能清除）
  - 藤壶 → 刮刀（爽快的"噶"声，一片一片剥落）
  - 珊瑚 → 高压水枪（大面积清除，有水流粒子特效）
- 工具可升级：刷子 → 工具箱 → 超声波清洗仪 → 自动化流水线
- 自动化出现后，普通品质文物可委托"学徒"后台处理，玩家仅需亲自处理稀有文物

**阶段 2：精细修复（拼图小游戏）**
- 破损等级较高的文物（稀有品质以上）需要额外的"拼图修复"步骤
- 玩家将碎片拖拽对齐到正确位置
- 对齐越精准，最终估值越高（完美修复 vs 有明显痕迹）
- 某些文物有"缺失碎片"——碎片分散在不同打捞地点，需要多次出海才能集齐

**阶段 3：专家鉴定（真伪对决）**
- 清洗完成，邀请周教授鉴定
- 一个简短的"鉴定问答"小游戏（3 道选择题）：
  - 题目来源于玩家在清洗过程中观察到的细节（釉色、花纹、落款）
  - 答对越多，鉴定价值越高，并解锁该文物的"历史故事"（可读可不读）
  - 答错导致误判，会影响后续交锋阶段的话语权

---

### 幕三：交锋（Confrontation）

> **MVP 阶段可简化为纯文字选择；中期迭代为竞价系统；远期迭代为卡牌博弈**

当你的声望积累到一定程度，三方势力开始对你的收获虎视眈眈：

**势力A — "陈先生"（黑市中间人）**
文质彬彬，出价永远比市价高 20%。他背后是谁，你不知道，但你能猜到。

**势力B — "淘宝军团"（竞品商家）**
会抢占你发现的打捞点，降低该区域的文物品质，并在网上给你的铺子刷差评。

**势力C — "收藏家协会"（表面正规）**
以"文化保护"为名，试图通过正规渠道将文物纳入私人收藏。

**交锋形式（三选一，视情境触发）**：
- **竞拍**（竞价）：三方竞价，玩家用资金+特殊"底牌"（舆论牌、鉴定书、关系牌）博弈
- **谈判**（剧情选择）：一次有时间限制的对话，选择不同回应路线，没有唯一正确答案
- **紧急撤离**（Arcade 小游戏）：躲避障碍物的竖屏快艇逃脱，保住手中的文物

---

### 幕四：抉择（Choice）

每件清理完成的文物，提供三条路径：

| 路径 | 收益 | 后续影响 |
|---|---|---|
| **卖给私人藏家** | 高金币 | 图鉴留白，声望小幅下滑，可能触发"后悔"支线 |
| **卖给国内文物市场** | 中等金币 | 文物进入民间，有机会后续回购（需更高价格） |
| **捐献给国家/博物馆** | 0 金币，高声望 + 荣誉勋章 + 全局 Buff | 文物陈列博物馆，点亮图鉴 |

**特殊机制：套装效应**
- 某些具有真实历史背景的文物被标记为"流失文物"系列（如圆明园系列）
- 集齐该系列时，系统锁定"出售"选项，仅允许"捐献"（这一设定在弹窗文字中被解释得自然而然，而非强制说教）
- 集齐并捐献整套后触发专属史诗动画，并永久解锁该系列的"历史长廊"（可阅读的沉浸式图文故事）

### 幕五：传承（Legacy）— 博物馆家园系统

**博物馆是这个游戏最终的"家"**，是玩家最愿意每天回来看一眼的地方。

#### 进化路径
```
陈记古玩铺（破烂小店）
  → 街边古玩店（有了展示柜）
    → 文物修复工作室（有了专业设备）
      → 私人小型博物馆（第一批游客）
        → 市级知名博物馆（媒体报道）
          → 国家认证博物馆（外公的愿望实现了）
```

#### 博物馆的日常
- **每日开馆**：自动产生"参观收入"（被动金币来源）
- **特别访客**：NPC 访客会触发随机支线任务（某小学生看到某件文物，开始哭泣——他的家族和这件文物有段历史……）
- **专题展览**：玩家可以手动策划一次"特展"，把相关文物集中陈列，吸引大量游客，短期大幅提升收入和声望
- **图鉴收集**：博物馆墙上有一张"流失文物地图"，空格越来越少，是最强的长线驱动

#### 大结局触发条件
当图鉴完成度达到 70% 时，一封信从未知地址寄到博物馆。
是外公写的，日期是他"最后一次深潜"的前一天。
信中提到了一件你从未在外公手记中见过的文物——它在何处，外公为何而去，这，才是整个游戏真正想讲的故事。

## 五、数值成长与自动化节奏

避免《潜水员戴夫》后期"繁琐劳动"问题的核心解法：

| 游戏阶段 | 玩家核心体验 | 自动化程度 |
|---|---|---|
| 早期（0~5天） | 亲手刮每一件文物，真实感受手艺的质感 | 无自动化 |
| 中期（1~4周） | 学徒接手普通文物，玩家专注稀有文物和交锋 | 部分自动化 |
| 后期（1个月+） | 玩家是"馆长"，偶尔亲自出马处理传奇文物 | 高度自动化 |

**核心原则：手动操作不消失，只集中在真正值得的时刻。**

## 六、美术风格参考

- **整体基调**：温暖、做旧、有故事感。参考《风来之国》的手绘质感 + 中式水墨插画的留白美学
- **文物本体**：写实感，清洗前后的视觉反差要极其强烈（从破铜烂铁 → 金光熠熠）
- **UI 风格**：参考博物馆陈列卡片（米白底色、宋体标注、真实的文物尺寸/年代注释）
- **角色设计**：卡通化的小人物，不需要大制作，重在神态和性格
