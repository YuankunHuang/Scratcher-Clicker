# 核心玩法与题材选型思考（内部讨论版）

当前客户端基础设施（UI框架、事件总线、窗口管理等）已基本搭建完毕。接下来我会先进入**核心玩法（Core Gameplay）**的试开发阶段。

我们目前确定的核心底层交互是：**“刮刮乐/刮除涂层” + “点击/放置（Idle/Clicker）数值养成”**。
为了让底层机制具备极强的**成瘾性、解压感**和**合理的商业化空间**，我们需要为其套上一个合适的题材外壳。

目前我个人整理出2个比较有潜力的方向。请大家阅读以下分析，并给出倾向和建议。这不一定是最终决定，但将决定我们第一个 MVP（最小可行性产品）的开发方向。

---

## 候选方案 A：【古董修复 / 深海寻宝】

**核心概念**：玩家从深海打捞或跳蚤市场低价买入裹满泥沙/铁锈的“泥疙瘩”（盲盒）。通过物理刮擦清理出底下的物品，经专家鉴定后卖出赚钱或捐献国家。

### 1. 核心循环 (Core Loop)
- **动作层（解压）**：疯狂刮擦屏幕，清除泥土和藤壶，伴随清脆的沙沙声和碎片飞溅，底图逐渐显露。
- **数值层（养成）**：卖出修复好的古董获得资金 -> 升级清洁工具（小刷子升级为高压水枪、超声波清洗仪）和打捞深度 -> 自动清洁（雇佣学徒）。
- **目标层（长线）**：极小概率刮出“国宝（如商代青铜器）”。面临道德抉择：卖掉换取天价金币，或无偿捐献给国家以解锁“国家博物馆”的最高荣誉图鉴和全局 Buff。

### 2. 优势与劣势
- **优势 (Pros)**：
  - **立意高，具备传播性**：“为国寻宝”、“捐献国宝”的家国情怀极易在短视频平台（如抖音/视频号）引发情感共鸣和自发传播。
  - **美术上限高**：从破铜烂铁到绝世珍宝的视觉反差极其强烈，给美术极大的发挥空间。
  - **长线留存好**：“国家博物馆”的图鉴收集系统能有效拿捏玩家的收集癖和强迫症。
- **劣势 (Cons)**：
  - 美术资产需求量较大（需要绘制大量的古董图鉴及不同破损状态）。
  - 需要一定的文本和鉴定包装（真伪专家的台词）。

## 候选方案 B：【疯狂赌石 / 翡翠切割】

**核心概念**：玩家在原石市场购买不同场口的翡翠原石。通过擦皮（开窗）和切割，体验“一刀穷一刀富”的极致刺激，最终建立自己的玉石帝国。

### 1. 核心循环 (Core Loop)
- **动作层（刺激）**：用手指（砂纸/切刀）在石头上擦除表皮。擦出一点绿光，心跳加速。
- **数值层（暴富）**：切出极品翡翠（如帝王绿），估值瞬间暴涨百倍。交由师傅雕刻成成品，卖出海量钞票 -> 购买更贵的极品原石 -> 雇佣机器或徒弟自动开石。
- **目标层（长线）**：成为玉石大亨，盘下整个原石市场，享受每秒进账万亿的极度数值膨胀。

### 2. 优势与劣势
- **优势 (Pros)**：
  - **极度洗脑，变现直接**：完美切中下沉市场对财富的原始渴望，高风险高回报的“赌博”心理机制能在极短时间内提供爆棚的多巴胺。
  - **美术资产复用率高**：石头的表皮、内部的绿/白肉、雕刻成品可以模块化生成，美术工作量相对可控。
- **劣势 (Cons)**：
  - 题材相对下沉，可能被部分玩家认为“俗气”或有擦边球嫌疑（虽然虚拟经济合法）。
  - 缺乏深度的情感羁绊，玩家容易在数值膨胀到极限后感到空虚而流失。

## 核心技术难点（两者共通）

无论选择哪种题材，首要攻克的技术难点是一致的：
**丝滑且高性能的刮擦渲染器 (Scratch Renderer)**。
所以我后面要做的大部分事情不会受题材的影响。
