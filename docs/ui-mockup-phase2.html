Phase 2 · UI/UX 设计说明

色彩系统

  • 主色调：暖棕 #6B3A2A（古玩铺质感）
  • 强调色：博物馆金 #C9A84C
  • 背景纸色：#F2E8D9 / #FFF8EE
  • 稀有色：金 / 紫 / 绿（三级稀有度）

① 刮擦主界面

  • MainWindow 主游戏界面
  • 顶部：资源栏（金币 + 声望）
  • 文物名片：当前文物信息 + 稀有度标签
  • 中心：深色刮擦区，文物居中
  • 底部：进度条 + 工具选择 + 底部导航

② 鉴定弹窗

  • AppraisalWindow blurBackground=true
  • 进度达 90% 自动触发
  • 文物"大展示"卡片，带粒子特效
  • 估值 / 品相 / 清洁评分
  • 历史简注（可选读）
  • 卖出 / 捐献 两个行动按钮
  • 国宝级：卖出按钮置灰，仅可捐献

③ 工具升级

  • UpgradeWindow 底部滑出
  • 三张工具卡横排，当前高亮
  • 未解锁工具置灰 + 显示费用
  • 升级说明 + 一键升级按钮

④ 博物馆

  • MuseumWindow
  • 顶部：收藏进度条（n/30）
  • 日均收入 / 声望 / 已入馆 三项统计
  • 按历史时期分组，每组 3 列网格
  • 已入馆：展示文物图 + 名称 + 稀有点
  • 未集齐：锁图标 + "???" 黑色剪影

待补充的美术资产

  • 文物底图（脏 / 清洁两张，每件文物）
  • 笔刷工具图标（三档）
  • 金币 / 声望图标
  • 博物馆背景纹理
  • 稀有度边框贴图
  • NPC 立绘（Phase 3）
