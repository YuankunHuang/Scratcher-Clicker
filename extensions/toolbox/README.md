# Toolbox (Cocos Editor extension)

- **Enable** in **Extension Manager**, then **Refresh** if needed.
- **Panels → Tools Panel** opens the dockable toolbox.
- **Window Manager** — create / edit `@windowController` attributes / delete window folders and sync `WindowNames.ts` + `ui/windows/index.ts`.

After creating a window, add the prefab under `assets/resources/` at the path shown in the status message.
